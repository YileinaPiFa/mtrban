## 快速开始

### 克隆仓库

```bash
git clone https://github.com/YileinaPiFa/mtrban.git
```

### 进入项目目录

```bash
cd SearchForBanWeb
```

### 安装依赖

```bash
npm install
```

### 启动开发环境

```bash
npm run dev
```

默认访问：

```
http://localhost:5173
```

### 构建生产版本

```bash
npm run build
```

### 本地预览

```bash
npm run preview
```
